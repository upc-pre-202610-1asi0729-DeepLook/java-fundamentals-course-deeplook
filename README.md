## Curso de Fundamentos de Java - [DeepLook]

Bienvenido al repositorio oficial de nuestro curso de Fundamentos de Java. Este repositorio funciona como una **biblioteca de código fuente** de respaldo para las lecciones y prácticas del curso.

🚀 **Nota importante:** No necesitas descargar ni instalar nada en tu computadora. Todo el curso está diseñado para trabajarse de forma interactiva en entornos en línea como **OnlineGDB**.

---

## 📂 Estructura del Repositorio

El código está organizado en las siguientes carpetas para facilitar tu aprendizaje:

* **`/starter-files`**: Contiene las plantillas de código iniciales. Encontrarás la estructura base de los ejercicios para que los copies y los resuelvas en tu editor en línea.
* **`/completed-examples`**: Contiene los códigos completamente resueltos. Te servirán como guía de referencia para comparar tus soluciones con las del equipo.
* **`/scripts`**: Archivos de automatización y configuración interna para la gestión del curso (uso administrativo/profesores).

---

## 💻 ¿Cómo usar este código en OnlineGDB?

Si deseas practicar con alguno de los archivos de este repositorio, sigue estos sencillos pasos:

1. Navega en este repositorio y entra a la carpeta del ejercicio que te interesa (ya sea en `/starter-files` o `/completed-examples`).
2. Haz clic sobre el archivo `.java` que deseas utilizar.
3. En la esquina superior derecha del bloque de código, haz clic en el botón **"Copy raw contents"** (el icono de los dos cuadrados) para copiar todo el texto.
4. Abre [OnlineGDB](https://www.onlinegdb.com/) (asegúrate de seleccionar **Java** en la esquina superior derecha del editor).
5. Borra el código por defecto, **pega** el código que copiaste ¡y listo! Ya puedes presionar **Run** para ejecutarlo.
